# Memory Index

Index chargé à chaque session. Une ligne par memory, pas de contenu ici.
Format d'un fichier memory : frontmatter (`name`, `description`, `metadata.type`) + corps.
Types : `user` (qui est l'utilisateur), `feedback` (comment travailler, avec le pourquoi),
`project` (travail en cours), `reference` (liens externes).
